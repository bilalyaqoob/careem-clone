# expo-firebase-auth-starter

## Setup:

- Create a `constants/ApiKeys.js` file and put your actual firebase config values:
```
export default {
  FirebaseConfig: {
    apiKey:      "",
    authDomain:  "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: ""
  }
}
```
- Run `npm install`
